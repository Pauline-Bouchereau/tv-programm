### 🚀 Running the project

#### Clone this repository :

```javascript
git clone https://github.com/Pauline-Bouchereau/tv-programm
cd tv-programm
```

#### Install packages :

```javascript
yarn
```

#### When installation is complete :

```javascript
yarn start
```
